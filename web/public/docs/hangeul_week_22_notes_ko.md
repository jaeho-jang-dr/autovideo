## 1. 핵심 개념: 한글의 구조와 원리
### 1.1. 한글의 제자 원리와 모아쓰기
한글은 개별 음소를 나열하는 선형적 방식(알파벳)과 달리, 자음과 모음을 하나의 사각형 틀 안에 결합하여 음절 단위로 만드는 **'모아쓰기'** 체계를 가집니다.
*   **천지인(天地人) 원리:** 모음은 하늘(•), 땅(ㅡ), 사람(ㅣ)을 상징하는 세 가지 요소를 바탕으로 만들어졌습니다.
*   **음절 블록 구조:** 하나의 음절은 반드시 '자음 + 모음' 혹은 '자음 + 모음 + 자음'의 구조를 갖춥니다.
    *   **초성(1):** 음절의 시작 자음
    *   **중성(2):** 반드시 모음으로 구성
    *   **종성(3, 4):** 받침이라 불리는 끝 자음
### 1.2. 모음의 분류와 배치
모음의 형태에 따라 자음(초성)이 결합하는 위치가 결정됩니다.
| 모음 유형 | 해당 모음 | 결합 위치 |
| :--- | :--- | :--- |
| **수직 모음** | ㅏ, ㅑ, ㅓ, ㅕ, ㅣ, ㅐ, ㅔ | 자음의 **오른쪽** |
| **수평 모음** | ㅗ, ㅛ, ㅜ, ㅠ, ㅡ | 자음의 **아래쪽** |
| **복합 모음(L자형)** | ㅘ, ㅝ, ㅟ, ㅙ, ㅞ | 자음의 **아래와 오른쪽**을 감싸듯 배치 |
### 1.3. 음양의 조화와 모음 조화(Vowel Harmony)
한글 모음은 소리의 느낌에 따라 양성과 음성으로 나뉘며, 복합 모음을 형성할 때 같은 성질끼리 결합하려는 경향이 있습니다.
*   **양성 모음:** 밝고 산뜻한 소리 (ㅏ, ㅗ, ㅐ). 시각적으로 난색 계열(빨강, 주황, 노랑)로 표현 가능.
*   **음성 모음:** 어둡고 무거운 소리 (ㅓ, ㅜ, ㅔ). 시각적으로 한색 계열(파랑, 남색, 보라)로 표현 가능.
*   **중성 모음:** 사람을 상징하며 어디든 결합 가능한 'ㅣ'. 무채색이나 반투명 블록으로 시각화하여 조화를 강조.

## 2. 한국어 발음과 음운 규칙
### 2.1. 표준 발음법의 기초
한국어에는 21개의 모음과 19개의 자음이 존재합니다. 
*   **단모음(10개):** ㅏ, ㅐ, ㅓ, ㅔ, ㅗ, ㅚ, ㅜ, ㅟ, ㅡ, ㅣ (발음 시 입 모양이 변하지 않음).
*   **이중 모음(11개):** ㅑ, ㅒ, ㅕ, ㅖ, ㅘ, ㅙ, ㅛ, ㅝ, ㅞ, ㅠ, ㅢ (발음 시 입 모양이 변함).
*   **받침의 대표음:** 자음은 많으나 종성(받침) 위치에서 실제로 발음되는 소리는 **ㄱ, ㄴ, ㄷ, ㄹ, ㅁ, ㅂ, ㅇ**의 7개뿐입니다.
### 2.2. 주요 음운 변화 현상
자연스러운 한국어 구사를 위해 다음의 소리 변화 규칙을 숙지해야 합니다.
*   **연음 법칙:** 받침 뒤에 모음으로 시작하는 음절이 오면 받침 소리가 다음 음절의 첫소리로 옮겨 발음됨 (예: '읽어요' -> [일거요]).
*   **비음화:** 폐쇄음(ㄱ, ㄷ, ㅂ) 뒤에 비음(ㄴ, ㅁ)이 오면 각각 [ㅇ, ㄴ, ㅁ]으로 변함 (예: '국물' -> [궁물]).
*   **구개음화:** 받침 'ㄷ, ㅌ'이 조사나 접미사의 'ㅣ'와 만나면 [ㅈ, ㅊ]으로 변함 (예: '굳이' -> [구지]).
*   **된소리되기:** 예사소리가 특정한 환경에서 딱딱하고 빡빡한 느낌의 된소리(ㄲ, ㄸ, ㅃ, ㅆ, ㅉ)로 발음됨.

## 3. 실전 한국어: 자기소개와 기본 문법
### 3.1. 이름 및 국적 소개
처음 만났을 때 사용하는 가장 기본적인 의사소통 구조입니다.
*   **인사:** "안녕하세요?"
*   **지시 대명사:** "저는 ~이에요/예요."
    *   **이에요:** 명사에 받침이 있을 때 (예: "저는 학생이에요.")
    *   **예요:** 명사에 받침이 없을 때 (예: "저는 요리사예요.")
*   **국적 표현:** [나라 이름] + '사람' (예: "저는 한국 사람이에요.", "저는 미국 사람이에요.")
### 3.2. 주요 직업 어휘
*   학생, 회사원, 변호사, 가수, 의사, 경찰, 요리사, 운동선수.
