# 졸라 다이나마이트 — Google Flow/Veo 프로덕션 스토리보드 (재제작 v3)

> 사용자 요구 전면 반영. **반드시 Google Flow/Veo로 영상 생성.** 가이드 이미지 먼저 → 그 계획대로 씬 스크립트 → Veo 클립 → 컴파일 → **구글 드라이브 업로드.**

## 글로벌 규칙 (전 씬 공통)
- **배경**: 파르테논 신전 — **투명도 50%로 전체 화면 배경**(연하게 비치는 고전 신전). 모든 씬 동일 유지.
- **캐릭터**: 졸라맨 + 졸라걸, **2배 크게**(화면 비중 큼). 손그림 졸라 스타일 유지.
- **오프닝**: 둘이 **손을 잡고 시작**하는 이미지에서 출발.
- **글자(자모)**: 날아올 때 **크기 제각각·불규칙**, **여러 무리로 여러 차례** 등장, **빙빙 돌면서** 날아옴. 레고 블록이라 **색깔 제각각 OK**. **정렬할 때만 크기·각도가 같아짐.**
- **안무**: 드롭(음악 확 들어오는 순간)에 **임팩트 있는 오리지널 디스코·펑크 군무**(저작권상 특정 안무 복제 금지 → 동작은 오리지널, 타이밍·강도는 드롭에 정확히 일치).
- **클립 길이**: **3~8초 사이로 다양·불규칙**(정형적으로 안 느껴지게).
- **전환**: 음악 타이밍에 맞춰 컷. **연속이어야 하는 구간은 last image transition**(앞 클립 마지막 프레임을 다음 클립 첫 이미지로 업로드) 사용.
- 화면비: 9:16 세로(쇼츠). 텍스트/워터마크 없음(영어 라벨은 컴파일 단계 오버레이).

## 가이드 이미지 (Flow 이미지 모드로 먼저 생성)
| ID | 용도 | 내용 |
|----|------|------|
| G1 | S1 첫 프레임 | 졸라맨·졸라걸 **손잡고** 정면, 2배 크게, 파르테논 50% 배경 |
| G2 | S3 베이스 | 두 캐릭터 댄스 준비 자세 + 색색 자모블록 몇 개 공중 회전, 파르테논 50% |
| G3 | S4 첫 프레임(드롭) | 드롭 순간 **싱크 파워 포즈**(팔 위로/포인트), 역동적 |
| G4 | S6 베이스 | 자모가 두 줄(비티에스/다이나마이트)로 정렬되기 시작, 균일 크기 |

## 씬 스크립트 (Veo 클립)
| S | 길이 | 가이드/전환 | 내용 요약 |
|---|------|-------------|-----------|
| S1 | 6s | G1 / →last-image | 손잡고 미소 → 손 놓고 준비 자세. 카메라 천천히 푸시인. 파르테논 은은히. |
| S2 | 4s | last-image(S1) | 첫 자모 무리: 크기 제각각 색색 블록이 **오른쪽 위에서 소용돌이치며** 등장, 캐릭터 주위를 돌며 떠다님. |
| S3 | 7s | G2 / 컷 | 둘째·셋째 무리가 **다른 방향에서 빙빙** 더 날아옴. 공중에서 불규칙 군무. 캐릭터 댄스 빌드업. |
| S4 | 3s | G3 / 컷(온비트) | **드롭!** 둘이 동시에 시그니처 오리지널 무브를 샤프하게. 빠른 컷. |
| S5 | 8s | last-image(S4) | 군무 지속 + 자모들이 정렬 향해 스트리밍, 크기 균일화·각도 정립 시작. |
| S6 | 5s | G4 / 컷 | 자모가 **비티에스 / 다이나마이트** 두 줄로 스냅 정렬(균일·색은 유지). 캐릭터 신나게. |
| S7 | 4s | last-image(S6) | 단어 완성, 아래 **BTS / dynamite** 표시(오버레이), 둘이 다시 손잡고 피날레 포즈. |

- 길이 합 ≈ **37초**, 불규칙(6·4·7·3·8·5·4).
- last-image transition 구간: S1→S2, S4→S5, S6→S7. 온비트 컷: S2→S3, S3→S4, S5→S6.

## 파이프라인
1. Flow 이미지 모드로 **G1~G4 가이드 이미지** 생성/다운로드.
2. 가이드 기반 `zolla_dynamite_prompts.txt`(image::motion) 확정.
3. `autoveo_flow` 업로드-기반 생성으로 S1~S7 Veo 클립 다운로드(필요 구간 last-frame 추출 연결).
4. 컴파일(다양한 길이·전환) + 영어 라벨 오버레이 + 자작 BGM.
5. QA(왜곡·마주봄·정렬) → **구글 드라이브 업로드** → content.db 기록.
