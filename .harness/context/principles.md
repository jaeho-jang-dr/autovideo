## P1 — 인코딩 (최우선)

**모든 텍스트 파일은 UTF-8 without BOM으로 저장한다.**

```
금지: PowerShell Set-Content -Encoding UTF8  (BOM 포함!)
허용: Python open(..., encoding='utf-8', newline='')
```

## P8 — Veo 및 Google Flow 비디오 제작 워크플로우 준수

**정적 이미지 줌인(Ken Burns) 연출은 금지하며, 반드시 Veo 기반의 모션 비디오 클립을 제작해 합성한다.**

*   대본 5~6초당 한 프레임 이미지 생성 (프롬프트 뒤에 `No watermark, no text, no binary code, no letters, no signatures.` 명시하여 워터마크 원천 차단)
*   비디오 생성 프롬프트 설계 시, 단순 줌인/팬 무빙이 아닌 **3~4초 동안 다른 사물/장면 노출, 다차원 카메라 워킹, 오브젝트/인물 상호작용** 등 실제 물리 동작 변화가 유도되도록 구체적으로 작성
*   Google Flow / Veo를 이용해 3~4초 길이 비디오 클립(.mp4)을 제작해 `assets/videos/` 폴더에 배치
*   자세한 내용은 [VEO_WORKFLOW.md](file:///D:/Entertainments/DevEnvironment/autovideo/VEO_WORKFLOW.md) 참조

## P9 — 비디오 오디오 및 렌더링 디폴트 규칙 (실수 방지 체크리스트)

*   **나레이션 속도 디폴트:** 항상 원래 나레이션 음성 속도 대비 **10% 빠르게 (1.1배속, `MultiplySpeed(1.1)` 적용)** 렌더링을 진행한다. 이에 맞춰 비디오 속도도 1.1배속으로 자동 가속 매칭된다.
*   **나레이션 성별 디폴트:** 항상 명확하고 신뢰도 높은 **여성 음성**(`gTTS` 한국어 기본 목소리)을 사용한다.
*   **TTS 캐시 불일치 버그 방지:** 대본 변경 시 구버전 오디오 캐시가 사용되어 자막과 엇갈리는 현상을 막기 위해, 렌더링 실행 전 반드시 기존 오디오 캐시 파일(`assets/audio/scene_*.mp3`)을 **강제로 삭제**하거나, 스크립트가 실행 시 무조건 오디오 파일을 덮어쓰도록(Overwrite) 한다.
*   **자막 폰트 절대 경로화:** Windows 환경에서 한국어 폰트가 깨져서 블록 표시(`☒☒☒☒`)로 나오는 에러를 방지하기 위해, 자막 폰트 경로는 반드시 윈도우 시스템 절대 폰트 경로인 **`C:\Windows\Fonts\malgun.ttf`**를 하드코딩하여 사용한다.
*   **Claude CLI 동기식 렌더링 완료 대기:** 백그라운드로 렌더링을 지시할 때 CLI 세션이 조기 퇴근하여 파이썬 프로세스가 폭사하는 문제를 방지하기 위해, 명령에 반드시 "파이썬 실행 완료 및 동영상 생성이 성공할 때까지 대기(Wait)"하도록 지시하거나 직접 파이썬 프로세스를 동기적으로 실행하고 감시한다.
