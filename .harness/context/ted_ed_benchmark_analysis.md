## 2. 제작 "엔진" — 핵심 인사이트

### A. 단일 엔진은 없다 (가장 중요한 사실)
TED-Ed는 **하나의 소프트웨어나 자동화 엔진으로 영상을 찍어내지 않는다.** 핵심은 **"전 세계 프리랜서 애니메이터·독립 스튜디오 네트워크"** 라는 인적 파이프라인이다.

- **모델:** TED-Ed는 인하우스 **편집/디렉팅 팀**(애니메이션 디렉터 — 예: Biljana Labovic, Lisa LaBracio, Jeremiah Dickey)을 두고, 실제 작화는 **건별로 외부 프리랜서/스튜디오에 발주**한다.
- **결과:** 매 영상마다 작화 스타일이 다르다(컷페이퍼, 셀 애니메이션, 스톱모션, 3D, 클레이, 모션그래픽 등). "TED-Ed스러움"은 그림체가 아니라 **편집 템포·내레이션·사운드·타이포그래피의 일관성**에서 나온다.
- **작화 툴:** 시그니처 2D 핸드드로잉은 **TVPaint**를 많이 쓰지만, 작가마다 After Effects, Cinema 4D, Blender, 실물 스톱모션 등 **각자 도구**를 사용. 즉 "엔진 = 사람 + 각자의 툴".

### B. 1편당 제작 워크플로 (약 4개월+)
1. **피칭** — 교육자가 레슨 아이디어 제출 (또는 내부 발굴)
2. **개발 미팅** — 주제 선정·각도 결정
3. **스크립트** — **800~900 단어**의 "간결하고 직접적인" 대본 (말로 다 설명하지 않고 그림이 보여주도록 설계)
4. **팩트체킹** — TED-Ed 자체 검수
5. **내레이터 + 애니메이터 배정** — 주제에 맞는 작가 매칭
6. **스토리보드 → 애니매틱 → 작화** — 다단계 리비전, 원작 교육자도 드래프트 검토
7. **사운드 디자인 + 음악 + 자막 + 공개**

- **비용 구조:** 교육자는 **무보수**(감사의 의미로 기프트카드). 핵심 비용은 **프리랜서 작화·내레이션·음악**. 비영리(TED) 모델이라 광고가 아닌 후원/브랜드 협업이 주 재원.
- **품질 vs 속도:** 1편에 4개월 = 극단적 고품질·저속도. 그래서 연 100편이 한계.

## 4. 성공 요소 분석 (왜 통하는가)

1. **"작은 질문, 큰 답" 공식** — 제목은 누구나 궁금한 일상 질문, 내용은 깊은 지식. 클릭 동기 + 학습 만족 동시 충족.
2. **5분 단위 호흡** — 수업 한 토막 길이. 교실·자투리시간 소비에 최적화.
3. **스타일 다양성 + 톤 일관성** — 영상마다 그림체는 달라도 "차분한 내레이션 + 미니멀 어쿠스틱 BGM + 우아한 자막"이 브랜드를 묶음.
4. **교육 생태계 락인** — 단순 영상이 아니라 ed.ted.com에서 퀴즈·토론·심화자료가 붙은 "레슨". 교사가 수업 도구로 채택 → 안정적 트래픽.
5. **비영리 권위** — TED 브랜드의 신뢰도. 팩트체킹된 콘텐츠라는 인식.
6. **에버그린** — 시사가 아닌 원리·역사·과학 중심이라 5년 뒤에도 조회수 발생(누적 50억 뷰의 비결).

## 5. 내가 따라하는 법 — autovideo 파이프라인 매핑

| TED-Ed 요소 | TED-Ed 방식 | 우리(autovideo) 대응 |
| :--- | :--- | :--- |
| 작화 | 프리랜서 핸드 애니메이션 (4개월) | **Veo / Google Flow 모션 클립** (`assets/videos/scene_X.mp4`) — 수 시간 |
| 스크립트 | 800~900단어, 간결·직접 | `SCENES` 리스트, 영상이 보여주는 구조 (Ken Burns 금지 규칙과 일치) |
| 내레이션 | 차분한 영어 스토리텔링 130~140 WPM | gTTS 한국어 여성 음성 1.1배속 |
| 사운드 | 어쿠스틱 미니멀 BGM −18~−22dB + 폴리 SFX | BGM 주파수 격리 + 0.1초 SFX 믹싱 |
| 자막 | 박스 없이 stroke/shadow, 하단 여백 | malgun.ttf, stroke 1.5, 충분한 마진 |
| 편집 | 모핑·패럴랙스·연속 패닝 (디졸브 금지) | Veo 프롬프트에 카메라 워킹·물리 동작 상세 지시 |
| 구조 | Hook(12%) → 본론 시대순(73%) → 통찰(15%) | 스토리텔링 템플릿 그대로 적용 |

**즉, 우리는 이미 TED-Ed의 "감성 레시피"를 역공학해 두었고, 차이는 작화 엔진을 사람→AI(Veo)로 바꾼 것.** 따라할 핵심:
- 제목은 **의문형 일상 호기심**으로.
- 1편 4~6분, 800~1,000단어(한국어는 1,200~1,500자) 대본.
- 에버그린 주제 선정(시사 X, 원리·역사·과학 O).
- 영상마다 스타일이 달라도 **내레이션·BGM·자막 톤은 고정**해 채널 정체성 유지.

## 6. 창의적으로 차별화하는 법 — AI 시대의 비대칭 우위

TED-Ed의 약점은 곧 우리의 기회다.

| TED-Ed의 한계 | 우리의 기회 (AI 레버리지) |
| :--- | :--- |
| 연 100편이 물리적 한계 (4개월/편) | **AI로 주 5~10편+ 가능 → 롱테일 주제 대량 커버** |
| 영어 중심 | **한국어 네이티브 채널 → 국내 교육시장 선점** |
| 2D 일러스트 한정 | Veo로 **실사급·시네마틱·SF 비주얼**까지 확장 |
| 일반 교양 | **틈새·심화·로컬 특화**(한국사, 수능 개념, K-과학 등) |
| 정적 5분 포맷 | **Shorts/릴스 동시 제작**으로 유입 깔때기 구성 |

**창의 전략 3가지:**
1. **속도로 깊이 파기** — TED-Ed가 "용 신화" 1편 만들 때, 우리는 "세계 각국 용 신화 12편 시리즈"를 같은 기간에. 롱테일 검색 트래픽 독식.
2. **로컬라이즈 + 재맥락화** — TED-Ed의 글로벌 주제를 한국 관점/사례로 재해석(예: "복리의 마법"을 한국 부동산·연금 사례로).
3. **하이브리드 비주얼** — 컷페이퍼 감성(역공학 문서)을 유지하되, Veo의 실사 모션을 섞어 TED-Ed가 못 만드는 "움직이는 다큐멘터리" 질감.

## 7. 내가 만들 수 있는 채널 분류 (Channel Catalog)

12개 주제 × AI 제작 강점을 교차해 도출한 채널 후보. **티어 = 진입 난이도/차별화 매력.**

### 🟢 Tier 1 — 바로 시작 추천 (수요 크고 AI 적합)
1. **"5분 과학 호기심"** — 생활 속 why(왜 하늘은 파란가, 왜 멍이 드는가). TED-Ed 직계. 에버그린·검색 강함.
2. **"한국사 다시보기"** — 사건·인물을 시네마틱 Veo로 재현. 로컬 독점 시장. 수능/교양 수요.
3. **"돈의 원리(경제·금융 교양)"** — 복리, 인플레이션, 보험을 한국 사례로. 광고단가 높음.
4. **"마음의 과학(심리·뇌)"** — 인지편향, 습관, 번아웃. 자기계발 수요 + 에버그린.

### 🟡 Tier 2 — 차별화 강함 (틈새·심화)
5. **"세계 신화·전설 지도"** — 문화권별 신화 시리즈. 비주얼 임팩트 최고(Veo 강점).
6. **"철학 한 입"** — 트롤리 딜레마, 영생, 자유의지. TED-Ed 인기 검증된 포맷.
7. **"우주·지구 스케일"** — 블랙홀, 심해, 지질시대. 시네마틱 비주얼이 곧 경쟁력.
8. **"수학이 보이는 순간"** — 개념 시각화(미적분·확률 직관). 제작 난이도↑이나 진입장벽=해자.

### 🔵 Tier 3 — 실험적/창의적
9. **"만약에(What-If) 다큐"** — 가상 시나리오를 실사급 모션으로. AI 비대칭 우위 극대화.
10. **"사라진 것들의 역사"** — 멸종동물·고대문명·옛 직업. 다큐 질감 + 향수 자극.
11. **"미래 보고서"** — 기후·AI·우주이주 등 미래 시나리오. Veo SF 비주얼.
12. **"몸의 안쪽(의학·해부 교양)"** — 면역, 통증, 노화. 건강 = 2025 최고 인기 카테고리.

> **추천 진입 순서:** Tier 1에서 **1개 채널을 골라 주 2~3편**으로 시작 → 시그니처 톤(내레이션·BGM·자막) 고정 → 30편 누적 후 데이터 보고 Tier 2 확장. TED-Ed가 14년 걸려 만든 라이브러리를, **AI 속도로 1~2년에 압축**하는 것이 핵심 전략.
