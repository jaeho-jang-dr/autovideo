# 수면의 과학 (Sleep Science) 하이브리드 제작 가이드

본 에피소드는 **Google Flow(AI 에셋 생성) + MoviePy(뼈대 컴파일) + CapCut 데스크톱(다이나믹 연출 및 디테일)**을 융합한 **하이브리드 비디오 파이프라인**의 첫 번째 시범 모델입니다.

---

## 1. 프로젝트 파일 구성
- `scenario.txt`: 에피소드 전체 대본 및 씬 정의 (Scene 0 ~ Scene 4)
- `sleep_prompts.txt`: Google Flow 비디오/이미지 생성을 위한 2D Digital Cutout 스타일 전용 프롬프트 리스트
- `finalize.py`: 1차 결합 및 자막 CC 임베딩 자동 컴파일용 스크립트

---

## 2. 하이브리드 제작 워크플로우 (4단계)

### [1단계] 에셋 배치 생성 (AI 엔진)
로컬 터미널에서 `autoveo_flow.py`를 실행하여 프롬프트 리스트에 있는 2D 일러스트 씬들을 개별 생성 및 다운로드합니다.
```powershell
# 예시: 0번 씬 이미지 생성 및 다운로드
python autoveo_flow.py --prompts sleep_science/sleep_prompts.txt --video-dir sleep_science/ --scene 0
```
* **결과물 위치 매핑**:
  - `sleep_science/scene_0.png`: 썸네일 및 인트로용 베이스 이미지
  - `sleep_science/scene_0.mp4` ~ `scene_4.mp4`: 각 씬별 모션 비디오 클립

---

### [2단계] 뼈대 컴파일 (MoviePy 자동화)
에셋 다운로드가 완료되면 `sleep_science` 디렉토리 아래의 `finalize.py`를 실행하여 1차 완성본 비디오를 인코딩합니다.
```powershell
python sleep_science/finalize.py
```
* **컴파일러 자동 처리 사항**:
  - `Scene 0` (인트로) 자동 맨 앞 결합 및 자막 싱크 offset 정렬
  - `scene_0.png`를 활용해 sprout green 한글 박스가 오버레이된 **유튜브 썸네일(`scene_0_thumbnail_korean.png`) 자동 생성**
  - 바텀 영어 자막 번인 해제 및 **ko/en 토글형 소프트 CC 자막 자동 내장(Mux)**
  - 4초 단축 아웃트로(종소리 마감) + **10초 무음 최종 정지 화면(Outro Card) Concat 결합**

---

### [3단계] 캡컷(CapCut) 데스크톱 튜닝 (다이나믹 연출)
1차 컴파일 완료된 `sleep_science.mp4` 또는 개별 클립 소스들을 캡컷 데스크톱 앱으로 임포트하여 최종 다이나믹을 부여합니다.
1. **AI 배경 제거 (Cutout)**:
   - 시계나 캐릭터 레이어를 복제한 뒤, 캡컷의 **'오토 컷아웃(배경 제거)'** 기능을 사용해 2D 컷아웃 객체로 분리합니다.
2. **키프레임 메트로놈 애니메이션**:
   - 컷아웃된 알람 시계나 지휘자의 팔에 시간차 **회전(Rotation) 키프레임**을 적용해 TED-Ed 특유의 탁탁 끊기는 유기적인 움직임을 연출합니다.
3. **자막 튕기기 효과**:
   - 캡컷 텍스트 템플릿의 다이나믹 애니메이션(글자가 튕겨 나오거나 회전하며 등장하는 효과)을 자막 트랙에 적용합니다.
4. **다이나믹 사운드(SFX) 및 오디오 믹싱**:
   - 알람이 폭발하는 구간에 캡컷 내장 효과음(충격음, 콰광 사운드)을 더하고, 뇌 내부 펄스 씬에서는 웅장한 신디사이저 펄스음을 추가해 생동감을 높입니다.

---

### [4단계] 유튜브 최적화 출력 (4K 업스케일)
편집이 끝난 영상은 캡컷 내보내기(Export) 시 **해상도를 4K(3840x2160)로 강제 업스케일링**하여 출력합니다.
- **이유**: 1080p로 업로드하면 유튜브에서 저품질 코덱(AVC1)을 부여해 선과 글자가 뭉개지지만, 4K로 업로드하면 최고 품질 코덱(VP09)이 자동 부여되어 2D 그래픽 라인이 칼같이 선명하게 유지됩니다.
