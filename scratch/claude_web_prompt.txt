[프로젝트: autovideo, 스택: Python + MoviePy + gTTS, Astro]
[작업 목표]:
1. 데이터베이스 익스포터인 'channel/export_web.py'를 수정하십시오.
   - 현재 에피소드 정보만 추출하고 있고 세부 대본인 'scenes' 테이블 정보가 content.json에 누락되어 있습니다.
   - 각 에피소드 아래에 'scenes' 리스트(seq, script_kr, script_en, duration_sec 포함)가 중첩되도록 쿼리를 보강하여 'web/src/data/content.json'으로 익스포트하게 수정하고 이를 실행하여 데이터를 추출하십시오.
2. 'web/' 디렉토리에 Astro 5.x 정적 웹사이트를 구축하십시오.
   - 'npx -y create-astro@latest ./web --yes --no-git --install' 명령어로 프로젝트를 초기화하십시오.
   - 'astro.config.mjs'에 다국어(i18n) 설정을 구성하십시오. (defaultLocale: 'ko', locales: ['ko', 'en'], routing: { prefixDefaultLocale: false }).
   - 라우팅은 한국어 홈('/', '/ko'), 영어 홈('/en'), 카테고리별 목록('/category/[category]', '/en/category/[category]'), 에피소드 상세('/lesson/[code]', '/en/lesson/[code]')로 구현하십시오.
3. 럭셔리 다크 UI 테마 디자인을 퍼블리싱하십시오.
   - 'src/styles/global.css'에 변수 기반으로 색상(딥블랙/딥네이비 HSL배경, 네온 바이올렛/블루 그라디언트 테두리 발광, 글래스모피즘 효과 카드 등)을 세팅하십시오.
   - Inter, Outfit, Noto Sans KR 웹폰트를 연동하십시오.
   - 호버 액션 등 인터랙션 요소를 적용하십시오.
4. 에피소드 상세 학습 화면 구현:
   - 상단에 YouTube 임베드 iframe 뷰어를 구현하십시오. (유튜브 ID가 없을 경우 대비한 대체 썸네일 노출 필수)
   - 하단 레이아웃에 좌우 2컬럼(Side-by-Side) 형태의 한/영 대본 대조 학습 뷰어를 구현하십시오. (각 씬의 seq 번호와 script_kr, script_en이 서로 라인 정렬되어 읽기 편해야 함)
5. 빌드 및 작동 확인:
   - 'npm run build'가 에러 없이 작동하는지 빌드하여 정상 완료되는 것을 확인하십시오.
