# 한글 초급 3주차 — 거센소리와 된소리 (Aspirated & Tense Consonants)

> 훈민정음/Sori Hangeul 한글 교육 커리큘럼 초급 3주차 학습 자료. (Dr. Jay 채널)
> NotebookLM 팟캐스트·노트·요약 생성용 소스. 한국어 학습자(외국인/초급) 대상.

## 1. 오늘의 주제
한국어 자음에는 세 갈래의 소리가 있다.
- **예사소리(평음)**: ㄱ ㄷ ㅂ ㅈ ㅅ — 부드럽고 편안하게 내는 기본 소리.
- **거센소리(격음, aspirated)**: ㅋ ㅌ ㅍ ㅊ — 입에서 공기를 강하게 내뿜는 소리.
- **된소리(경음, tense)**: ㄲ ㄸ ㅃ ㅆ ㅉ — 목(성대)에 힘을 꽉 주고 단단하게 내는 소리.

## 2. 거센소리 — 가획의 원리
거센소리는 예사소리에 **획을 하나 더해** 만든다(가획의 원리).
- ㄱ → ㅋ, ㄷ → ㅌ, ㅂ → ㅍ, ㅈ → ㅊ.
소리: 입안에 모은 공기를 터뜨리듯 **강하게** 내보낸다. 뜨거운 숨을 "후—" 하고 내쉬는 느낌. 입 앞에 손바닥이나 휴지를 대면 바람이 세게 느껴진다.
- 예시 단어: 코, 타조, 포도, 치마.

## 3. 된소리 — 쌍자음의 원리
된소리는 **같은 예사소리를 둘 겹쳐(쌍자음)** 만든다.
- ㄱ+ㄱ → ㄲ, ㄷ+ㄷ → ㄸ, ㅂ+ㅂ → ㅃ, ㅅ+ㅅ → ㅆ, ㅈ+ㅈ → ㅉ.
소리: 목구멍을 닫듯 힘을 주고, 공기가 새어나가지 않게 **꽉** 잡았다가 단단하게 터뜨린다. 입술/혀는 거의 안 움직이고 목에 힘이 들어간다.
- 예시 단어: 까치, 꼬리, 땅, 빵.

## 4. 휴지(종이) 실험 — 핵심 구별법
입 앞에 얇은 종이(휴지)를 대고 발음한다.
- **거센소리(ㅋ)**: 공기를 확 뿜어 종이가 크게 **흔들린다**.
- **예사소리(ㄱ)·된소리(ㄲ)**: 공기가 약해 종이가 **거의 안 흔들린다**.
이 차이가 거센소리와 나머지를 가르는 가장 직관적인 신호다.

## 5. 최소대립쌍 — 예사·거센·된 삼총사
같은 자리, 다른 세기. 소리만 다르고 뜻이 완전히 달라진다.
- ㄱ – ㅋ – ㄲ : **개**(dog) – **캐**(dig) – **깨**(sesame/break)
- ㅈ – ㅊ – ㅉ : **자**(ruler) – **차**(tea/car) – **짜**(salty)
- 발음 요령: 개=편안하게, 캐=공기를 힘껏, 깨=목에 힘주고 단단하게.

## 6. 정리
- 거센소리 = **획 추가** + **공기 셈**(숨이 강함).
- 된소리 = **쌍자음** + **목 긴장**(숨이 약함).
- 구별 도구: 휴지 실험, 최소대립쌍(개-캐-깨) 듣고 받아쓰기.
- 매일 조금씩 소리 내어 따라 읽으면 금방 익는다.
