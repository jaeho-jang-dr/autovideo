# 작업 지시 (최우선) — 감독(Claude) → 조감독(Gemini): NotebookLM 한글교육 노트북 전체 자료 정리

## 목표
한글교육 NotebookLM 노트북에 최근 **52개 소스가 추가**되었다. 이 노트북의 **모든 소스 자료를 빠짐없이 정리·요약**해서,
24주(초·중·고급) × 하루 1강 × 7일 = **168강 커리큘럼 설계의 근거 자료**로 쓸 수 있게 만들어라.
이게 이번 작업에서 **가장 중요한 일**이다.

NotebookLM 노트북 id: `cc6092e5-3322-44e8-b65e-dc0e85c2e3ed`. notebooklm-mcp 도구 연결됨(MCP 도구는 사용 가능, 셸은 막혀 있을 수 있음 — 셸은 감독이 처리).

## 할 일 (notebooklm-mcp MCP 도구 + write_file 사용)
1. 노트북의 **전체 소스 목록**(모든 소스 제목)을 가져와라 (`notebook_get` 또는 소스 목록 도구).
2. 소스들을 **난이도(초급/중급/고급)와 주제별로 분류**하고, 각 소스가 가르칠 수 있는 **핵심 포인트**(어휘·문법·발음·문화 등)를 정리하라.
   - 가능하면 각 소스가 아래 24개 주차 중 **어느 주제에 해당하는지** 매핑하라:
     (초급) 1단모음 2자음 3거센·된소리 4받침 5이중모음 6연음 7인사·자기소개 8숫자·시간
     (중급) 9동네·위치 10쇼핑 11식당 12교통 13길찾기 14하루일과 15날씨 16취미
     (고급) 17K컬처 18감정 19의견·설득 20돌발상황 21인물묘사 22여행 23모임 24마무리
   - 결과를 `scratch/nlm_korean_digest.md` 에 **write_file** 로 작성하라. 형식:
     `## [난이도/주차] 주제` 아래에 `- 소스제목 — 핵심 가르칠 점` 목록.
3. 추가로, 노트북 **전체 소스를 종합한 스터디 리포트**(report)를 한국어 1개 생성(`studio_create`/`create_report`)하고,
   생성한 artifact_id 를 `scratch/gemini_nlm_digest_report.md` 에 기록하라(감독이 다운로드함).
4. 막히면(도구 권한 등) 그 사실을 `scratch/gemini_nlm_digest_report.md` 에 적고, 할 수 있는 데까지 진행하라.
