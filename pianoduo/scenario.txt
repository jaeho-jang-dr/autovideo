#
# ★ 절대 규칙 (사용자 피드백):
#   1) 두 사람은 항상 같은 벤치에 '나란히' 앉는다. 절대 서로 마주 보지 않는다(물리적으로 불가).
#   2) 연주 구간(2~24)에서 두 사람(특히 지은)의 손은 '항상 건반 위'에 있고 실제로 계속 누르며 친다 — 소리=손동작 일치.
#   3) 표정은 진지하고 집중. 연주 중 과한 웃음 금지. (환한 웃음은 인사 1씬 · 작별 25씬에서만.)
#   4) 첫 음: 손가락이 건반에 닿는 순간과 첫 소리가 '정확히 동시에' 시작.
#   5) 끝 음: 마지막 화음 소리와 동시에 손이 건반에 닿고, 그대로 건반 위에서 '멈춤'을 보여준다.
#   6) 인준은 적당한(controlled) 퍼포먼스 — 가끔 일어서되 손은 항상 건반 위. 연탄곡이라 두 사람 동작이 같을 필요는 없다.
#   7) 카메라: 지미집처럼 부드럽고 역동적인 시점 이동(상승/하강/푸시인/궤도/탑다운/손 줌인). ※장비 자체는 화면에 절대 안 보임.
